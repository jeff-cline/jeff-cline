# Jeff Cline Business Dashboard
## Agency Platform v1.0 | "A Rising Tide Lifts All Boats"

### Quick Start
1. Unzip the folder
2. Open `index.html` in any browser (public landing page)
3. Open `dashboard.html` for the full agency dashboard
4. No server required - runs as static files

### Files
```
jeff-cline-dashboard/
├── index.html          → Public landing page (SEO/AEO optimized)
├── dashboard.html      → Full agency dashboard (17 views)

├── js/app.js           → Dashboard core logic
├── js/charts.js        → Chart.js visualizations

```
